In the directory 'test-cases/case3/', create a new subdirectory named 'src'.
Inside this new 'test-cases/case3/src/' subdirectory, create a new TypeScript file named 'utils.ts'.

In this 'test-cases/case3/src/utils.ts' file, define and export the following two functions:

1.  A function named `isNullOrEmpty` that takes one argument `value` of type `string | null | undefined`.
    It should return `true` if the `value` is null, undefined, or an empty string (""). Otherwise, it should return `false`.

2.  A function named `capitalizeFirstLetter` that takes one argument `str` of type `string`.
    It should return a new string with the first letter capitalized.
    If the input string is empty, or if the input string itself is null or undefined, it should return an empty string.
